Rhythm & Hues Studios

While a lead animator at Rhythm & Hues, I found time to write and support 20+ widely used tools for Voodoo, their amazing, proprietary animation software. Listed below are some of the favorites.

COG Record, edit and reapply character animation in worldspace with ease.
Snap Record, edit and reapply a single object's worldspace animation with ease.
Tweener Set in-between keys for your selected controls.
Buddies Establish spacial relationships between objects.
Control Tower Quickly select, edit, display and run commands on many characters at once.
Path Tools Simple tools for building and editing complex character paths.
